# HotWordsDisplayPlatform
HotWordsDisplayPlatform
基于web端的信息化领域热词展示平台

技术栈：HTML 5+NLP+ECharts+MySQL+爬虫
项目描述：从网络中爬取信息领域的相关热词，主要包括热词分类、热词查询、数据可视化展示、数据导出四方面的功能。
